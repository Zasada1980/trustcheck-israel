## 1. Общая информация

### Базовые данные
- **Официальное название:** Takdin Ltd. (часть Guideline Group Technologies)
- **Сайт:** https://www.takdin.co.il  
- **Материнская компания:** Guideline Group (владелец CheckID, Tabu.co.il, Nevo)
- **Год основания:** ~2000 (25+ лет присутствия)

### Категория
**Юридическая база данных (Legal Database)** — крупнейший агрегатор судебных решений в Израиле.

**Ключевое отличие:** Takdin специализируется **только на судебных данных** — не собирает финансовые/кредитные данные.

**Уникальная технология:** Naziclick (נזיקליק) — экономический анализ сумм исков и компенсаций.

### 2.2. Единственный государственный источник: נט המשפט (NetHaMishpat)

#### **Источник: Net HaMishpat (Court System)**

**Тип доступа:** ✅ Партнёрский Gateway Provider (официальный интегратор)

**Юридическая база:**
- Takdin имеет **официальный договор** с Courts Administration (Administration of Courts Law)
- Статус: **Gateway Provider** — один из 3-5 частных провайдеров с массовым доступом к NetHaMishpat

**Данные, получаемые Takdin:**

1. **Судебные решения (פסקי דין):**
   - Верховный суд (בית המשפט העליון)
   - Окружные суды (בתי משפט מחוזיים)
   - Магистратские суды (בתי משפט שלום)
   - Трудовые суды (בית דין לעבודה)
   - Семейные суды (בית משפט לענייני משפחה)
   - Военные суды (בית דין צבאי)
   - Религиозные суды (בית דין רבני) — частично

2. **Типы дел:**
   - Гражданские (תיקים אזרחיים)
   - Коммерческие (תיקים מסחריים)
   - Уголовные (תיקים פליליים)
   - Административные (תיקים מינהליים)
   - Трудовые (תיקים בעבודה)
   - Семейные (תיקים משפחתיים)
   - Банкротства (הדלות פירעון)

3. **Метаданные дел:**
   - Номер дела (מספר תיק)
   - Дата подачи иска (תאריך הגשה)
   - Стороны процесса (צדדים):
     - Истец (תובע)
     - Ответчик (נתבע)
     - Адвокаты (עורכי דין)
   - Судья (שופט/ת)
   - Статус дела (סטטוס תיק): открыто/закрыто/апелляция
   - Сумма иска (סכום התביעה) — если указана
   - Присужденная компенсация (פיצוי שנפסק)
   - Ключевые слова (מילות מפתח)

**Метод интеграции:**
```
[NetHaMishpat API] → Takdin Gateway Server (dedicated connection)
                   → Daily batch downloads (bulk XML/JSON)
                   → Takdin ETL Pipeline:
                      • Parse court decisions (OCR for scanned docs)
                      • Extract parties (NLP — Named Entity Recognition)
                      • Categorize cases (AI classification)
                      • Index for search (Elasticsearch)
                   → Takdin Database (2.5M+ decisions)
```

**Частота обновления:** 
- **Daily** — новые решения публикуются каждый день (обычно ночью)
- **Weekly** — массовое обновление метаданных
- **Monthly** — архивные дела (старше 10 лет)

**Покрытие:**
- ✅ **2.5M+ судебных решений** (с 1948 года)
- ✅ **95% всех опубликованных решений** NetHaMishpat
- ⚠️ **НЕ 100%** — некоторые решения засекречены (ביטחון מדינה, עניינים ביטחוניים)

### 2.3. ЧТО Takdin НЕ ИМЕЕТ (критично для SBF!)

**Критические ограничения:**

❌ **Companies Registrar (רשם החברות):**
- Takdin **НЕ интегрирован** с Companies Registrar
- Нет данных о владельцах, директорах, капитале
- Только упоминания компаний в судебных решениях

❌ **Bank of Israel Credit Registry:**
- Нет кредитной истории
- Нет Trade Payment Data
- Нет данных о платежах

❌ **Tax Authority (Shaam):**
- Нет налоговых данных
- Только налоговые споры (если дошли до суда)

❌ **Hotzaa LaPoal (Bailiff Authority):**
- Нет данных о должниках
- Только решения судов по исполнительным производствам

❌ **Land Registry (Tabu):**
- Нет данных о недвижимости
- Только земельные споры (если дошли до суда)

**Вывод:** Takdin — это **"зеркало бизнеса через суды"**, но НЕ полный источник бизнес-информации.

### 2.4. Уникальные технологии Takdin

#### **1. Naziclick (נזיקליק) — Экономический анализ исков**

**Описание:** AI-система для анализа сумм компенсаций в делах о телесных повреждениях (נזקי גוף).

**Функционал:**
- Поиск аналогичных дел по параметрам:
  - Возраст пострадавшего
  - Тип травмы (שבר, נכות, כוויה, etc.)
  - Степень инвалидности (אחוזי נכות)
  - Потеря трудоспособности
- Прогноз компенсации (prediction model)
- Статистика по судьям (какой судья присуждает больше/меньше)

**Применимость для SBF:** ⚠️ **Ограниченная** (релевантно только для трудовых/коммерческих споров, не для עוסקים)

---

#### **2. Techdin AI — GPT для юридических текстов**

**Описание:** Израильский аналог Harvey AI (юридический ChatGPT)

**Функционал:**
- Summarization (סיכום פסקי דין)
- Q&A по судебным решениям
- Legal research assistant
- Draft legal documents

**Применимость для SBF:** ❌ **НЕ релевантно** (SBF не генерирует юридические документы)

---

#### **3. Smart Agent (סוכן חכם) — Email Alerts**

**Описание:** Автоматические уведомления о новых судебных делах

**Функционал:**
- Пользователь создаёт "saved search" (חיפוש שמור):
  - Имя компании/физлица
  - Тип дела (גזר דין, פסק דין, החלטה)
  - Суд (בית משפט)
- Takdin отправляет email при появлении нового решения

**Применимость для SBF:** ✅ **Высокая** — SBF может предложить "мониторинг судебных дел" как premium feature

**Цена Takdin:** Включена в подписку (₪99-299/месяц)

## 3. Лицензии и разрешения

### 3.1. Gateway Provider License (רישיון ספק שער)

**Регулятор:** Courts Administration (המינהל לבתי המשפט)

**Юридическая база:**
- **Courts Law (1984)** — חוק בתי המשפט, תשמ"ד-1984
- **Freedom of Information Law (1998)** — חוק חופש המידע, תשנ"ח-1998

**Требования для получения:**
1. **Минимальный капитал:** ₪1,000,000
2. **Technical infrastructure:** Dedicated servers, secure connection
3. **Data security:** ISO 27001 certification
4. **Compliance:** Annual audit by Courts Administration
5. **Fee:** ₪50,000-200,000/год (в зависимости от объёма запросов)

**Статус Takdin:** ✅ Лицензия получена (~2000-2005, один из первых Gateway Providers)

**Права лицензии:**
- ✅ Массовый доступ к NetHaMishpat (bulk downloads)
- ✅ Real-time API к новым решениям
- ✅ Redistribution права (может продавать доступ клиентам)

**Ограничения лицензии:**
- ⚠️ Запрет на изменение текста решений (must preserve original)
- ⚠️ Обязан удалять решения по запросу суда (censorship)
- ⚠️ Не может публиковать засекреченные дела

**Конкуренты с аналогичной лицензией:**
- Nevo (Guideline Group) — конкурент, но та же компания-владелец
- PsakDin — независимый конкурент
- ERN (Electronic Registrar Network) — государственный портал (бесплатный, но ограниченный)

---

### 3.2. Database License (רישיון מאגר מידע)

**Регулятор:** רשות להגנת הפרטיות (Privacy Protection Authority)

**Статус:** ✅ Получена (обязательна для хранения данных о физлицах)

**Обязательства:**
- Ежегодная декларация в PPA
- Право физлиц на исправление данных (GDPR-like)
- Обязательное уведомление о data breach (72 часа)

### 3.3. Compliance & Security

- ISO 27001 (Information Security) ✅

**GDPR Compliance:** ✅ (судебные решения — публичные данные, но имена физлиц иногда анонимизируются)

3. **Налоговые дела** (תיקים מס הכנסה) → конфликт с Tax Authority
4. **Банкротство** (הדלות פירעון) → finality

Company X (H.P. 515123456):
