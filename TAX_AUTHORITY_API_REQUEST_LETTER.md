## מטרת הבקשה

אנו מבקשים לקבל **גישה ל-API של רשות המסים** לצורך בדיקת סטטוס עוסקים מורשים ופטורים, בדומה לשירות שניתן לחברות כגון CheckID (גיידליין גרופ).

## שימושים מתוכננים

1. **אימות סטטוס מע"מ** - בדיקה האם עסק רשום כעוסק מורשה או פטור
2. **אימות ניכוי במקור** - בדיקת רישום לניכוי במקור
3. **שקיפות עסקית** - מתן מידע מהימן לציבור לפני ביצוע תשלומים
4. **מניעת הונאות** - זיהוי עסקים בעייתיים לפני נזק כלכלי

## יתרונות לרשות המסים

- **הגברת שקיפות** - ציבור יוכל לאמת מידע עסקי בקלות
- **מניעת הונאות מס** - זיהוי עסקים לא רשומים
- **עידוד רישום** - עסקים יעודדו להירשם כדי להיות רשומים במערכת
- **שימוש אחראי** - מחויבים לשמור על rate limiting ונהלי אבטחת מידע

## מידע טכני

**פלטפורמה טכנולוגית:**
- Frontend: Next.js 14 (React)
- Backend: Node.js + PostgreSQL
- Cloud: Hetzner Cloud (Germany)
- Security: HTTPS, encryption at rest

**היקף שימוש צפוי:**
- Phase 1 (חודשים 1-3): ~1,000 שאילתות/חודש
- Phase 2 (חודשים 4-6): ~10,000 שאילתות/חודש
- Phase 3 (שנה 1): ~50,000 שאילתות/חודש

**Rate Limiting:**
מחויבים לשמור על rate limit של עד 100 שאילתות לדקה (או כפי שתקבעו).
