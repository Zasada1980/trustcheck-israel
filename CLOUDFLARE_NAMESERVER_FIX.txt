ИСПРАВЛЕНИЕ: Синхронизация Cloudflare с MyNames
═══════════════════════════════════════════════════════════════════

Проблема: Cloudflare ждёт jihoon/molly, MyNames настроил elsa/todd

Решение: Пересоздать домен в Cloudflare с правильными nameservers

═══════════════════════════════════════════════════════════════════
ШАГ 1: Удалить домен из Cloudflare
═══════════════════════════════════════════════════════════════════

1. В Cloudflare Dashboard → trustcheck.co.il
2. Прокрути вниз до "Advanced Actions"
3. Нажми: "Remove from Cloudflare"
4. Подтверди удаление

═══════════════════════════════════════════════════════════════════
ШАГ 2: Добавить домен заново
═══════════════════════════════════════════════════════════════════

1. Cloudflare Dashboard → "Add a Site"
2. Введи: trustcheck.co.il
3. Выбери план: Free
4. Cloudflare просканирует DNS (подожди 30 сек)

═══════════════════════════════════════════════════════════════════
ШАГ 3: Проверить DNS записи
═══════════════════════════════════════════════════════════════════

Cloudflare покажет найденные записи. Проверь/добавь:

Type    Name    Content             Proxy
A       @       46.224.147.252      🟠 ON
A       www     46.224.147.252      🟠 ON

Нажми "Continue"

═══════════════════════════════════════════════════════════════════
ШАГ 4: Получить НОВЫЕ nameservers
═══════════════════════════════════════════════════════════════════

Cloudflare покажет nameservers. ВОЗМОЖНО это будут:

elsa.ns.cloudflare.com     ← Совпадёт с MyNames!
todd.ns.cloudflare.com     ← Совпадёт с MyNames!

Если ДА → ОТЛИЧНО! Они совпадут с MyNames автоматически!

Если НЕТ (другие nameservers) → иди в MyNames и измени вручную

═══════════════════════════════════════════════════════════════════
ШАГ 5: НЕ МЕНЯТЬ nameservers в MyNames
═══════════════════════════════════════════════════════════════════

MyNames УЖЕ настроил elsa/todd.

Если Cloudflare выдаст ДРУГИЕ nameservers:
→ Придётся вернуться в MyNames → שרתי שם → поменять снова

Но СКОРЕЕ ВСЕГО Cloudflare выдаст elsa/todd (те же)!

═══════════════════════════════════════════════════════════════════
ШАГ 6: Cloudflare активируется автоматически
═══════════════════════════════════════════════════════════════════

Так как MyNames УЖЕ настроил nameservers:
→ Cloudflare сразу увидит изменения (5-15 минут)
→ Статус изменится на "Active"
→ Получишь email

═══════════════════════════════════════════════════════════════════
СЛЕДУЮЩИЙ ШАГ: SSL/TLS Configuration
═══════════════════════════════════════════════════════════════════

Когда Cloudflare покажет "Active":

1. SSL/TLS → Overview → Выбрать "Full (strict)"
2. Получить Origin Certificate от Cloudflare
3. Установить на сервере Hetzner
4. Обновить Nginx конфиг
5. Проверить https://trustcheck.co.il
